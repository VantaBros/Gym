# Installazione di VANTA 12

VANTA è una PWA e deve essere pubblicata su un indirizzo HTTPS, ad esempio tramite GitHub Pages.

## iPhone

1. Apri il link in Safari.
2. Tocca **Condividi**.
3. Seleziona **Aggiungi alla schermata Home**.
4. Conferma con **Aggiungi**.

## Android

1. Apri il link in Chrome.
2. Apri il menu con i tre puntini.
3. Seleziona **Installa app** o **Aggiungi alla schermata Home**.
4. Conferma.

## PC

Apri il link con Chrome o Edge e usa il pulsante di installazione nella barra degli indirizzi, oppure utilizza direttamente l’app nel browser.
