# VANTA 13

PWA personale per allenamenti con calendario selezionabile, Fantasma, catalogo esercizi, importazione universale e identità grafica ICE.

## Avvio
Pubblicare tutti i file su hosting HTTPS statico, mantenendo `index.html` nella cartella principale.

## Dati
Schede, calendario, sessioni e profilo sono memorizzati localmente nel browser. Eseguire periodicamente **Profilo → Esporta backup**.
